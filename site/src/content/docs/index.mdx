## What is Soul.md?

[Soul.md RFC-1](https://github.com/rokoss21/soul.md) is a portable, file-based
format for defining an AI agent's persona — its voice, values, interaction
style, safety posture, composition from reusable mixins, and reactive "mood"
states. muster is the reference **CTS-1** (Conformance Test Suite) harness for
that spec: it tests whether a SOUL.md file is *valid* and whether a model
*behaves* the way the file declares.

The normative spec text is vendored in the repository and is the single source
of truth for every check muster performs — each test cites the RFC-1 section it
enforces.
